# session and stream unlimited chunks without ever sending the final closing

# successor `asyncua`, so there is no upgrade that resolves it.

# demonstrates the unguarded library retaining 5000 chunks. Tunable via
# OPC_MAX_CHUNKS / OPC_MAX_MESSAGE_BYTES.
